## 📝 단계별 체크리스트

### **준비 사항 (지금 바로)**
- [ ] 신분증 스캔 준비 (주민등록증/운전면허증/여권)
- [ ] 전화번호 확인 (전화 인증용)
- [ ] 신용카드 준비 ($199 또는 $299)
- [ ] 이메일 확인: hocu00987@gmail.com

---

### **Step 1: 사이트 접속 (5분)**
- [ ] https://www.ssl.com/certificates/code-signing/ 클릭
- [ ] "Order Now" 버튼 클릭
- [ ] 제품 선택:
  - [ ] **eSigner EV Code Signing** ($299) ⭐ 추천!
  - [ ] 또는 eSigner Code Signing ($199)

---

### **Step 2: 계정 생성 (5분)**
- [ ] 이메일: hocu00987@gmail.com
- [ ] 비밀번호 설정 (안전하게!)
- [ ] 이름: [실명 입력]
- [ ] 전화: [휴대폰 번호]
- [ ] 국가: South Korea

---

### **Step 3: 조직 정보 (3분)**
- [ ] 조직명: QRChat (또는 개인 이름)
- [ ] 조직 유형: Individual
- [ ] 주소: [실제 주소]
- [ ] 도시/우편번호 입력

---

### **Step 4: CSR 제출 (이미 완료!)**
- [x] CSR 파일: `/home/user/qrchat_desktop/qrchat.csr`
- [ ] CSR 복사해서 붙여넣기 (구매시 요구됨)

```bash
cat /home/user/qrchat_desktop/qrchat.csr
```

---

### **Step 5: 결제 (2분)**
- [ ] 신용카드 정보 입력
- [ ] 결제 완료
- [ ] 주문 확인 이메일 수신 대기

---

### **Step 6: 검증 (1~3일)**
- [ ] **Day 1:** 검증 이메일 수신
- [ ] 신분증 스캔 업로드
- [ ] 전화 인증 대기
- [ ] **Day 2:** SSL.com에서 전화 받기
- [ ] **Day 3:** 인증서 발급 이메일 수신

---

### **Step 7: 인증서 다운로드**
- [ ] SSL.com 로그인
- [ ] "Download Certificate" 클릭
- [ ] `.pfx` 파일 다운로드
- [ ] 비밀번호 설정 & 메모
