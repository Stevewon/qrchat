QRChat v1.0.12 - 이미지/스티커 크기 일관성 수정
============================================

빌드 날짜: 2026-02-14
빌드 버전: v1.0.12 (Build 112)

📦 변경 사항
-----------
✅ 그룹 채팅방 이미지/스티커 크기 일관성 문제 수정
   - 채팅방 나갔다 들어와도 이미지/스티커 크기가 동일하게 유지됨
   - 스티커 크기: 75x75px (고정)
   - 일반 이미지 크기: 250x250px (고정 최대 크기)
   - MediaQuery 의존성 제거로 화면 크기 변화 시에도 일관된 표시

✅ 1:1 채팅방도 동일하게 수정
   - 모든 채팅 화면에서 일관된 이미지/스티커 크기 적용

🔧 기술적 변경사항
----------------
- LayoutBuilder 제거 (화면 크기 재계산 방지)
- 고정 크기 사용 (const double 값)
- Image.network에 명시적 width/height 지정
- 재진입 시에도 동일한 크기 보장

📱 설치 방법
----------
1. 기존 QRChat 앱 삭제 (데이터 보존을 원하면 Skip)
2. qrchat_v1.0.12.apk 설치
3. "알 수 없는 출처" 권한 허용 필요
4. 앱 실행 후 로그인

🧪 테스트 방법
------------
1. 그룹 채팅방 입장
2. 스티커 또는 이미지 전송
3. 뒤로가기로 나가기
4. 다시 채팅방 입장
5. 이미지/스티커 크기가 동일한지 확인 ✅
