# QRChat v1.0.16 설치 가이드

## 📱 다운로드

**APK 파일**: `qrchat_v1.0.16.apk` (70 MB)
**버전**: 1.0.16 (Build 116)
**빌드 날짜**: 2026-02-14

## 🔧 주요 수정 사항

### 1. ✅ QKEY 표시 텍스트 수정
- **이전**: "채팅 5분마다 10 QKEY 적립"
- **수정**: "채팅 5분마다 2 QKEY 적립"
- 실제로 2 QKEY가 지급되고 있었으나 UI 텍스트만 10으로 표시되던 문제 해결

### 2. ✅ QKEY 잔액 동기화 완료
- Firestore의 `qkeyBalance`와 `qkey_balance` 필드 동기화
- 모든 사용자의 잔액이 정확하게 표시됨
- 바보바보님: 60 QKEY (30건 × 2 QKEY)

### 3. ✅ 스티커팩 관리 기능 (이미 구현됨)
스티커팩 개별 관리 기능은 v1.0.11부터 이미 완전히 구현되어 있습니다!

**접근 방법**:
1. 프로필 탭 클릭
2. 우측 상단 ⚙️ (설정) 버튼 클릭
3. 설정 화면에서 스크롤 다운
4. "About" 메뉴 바로 아래에 **"스티커팩 관리"** 메뉴 클릭

**제공 기능**:
- ✅ 새 스티커팩 생성 (우측 상단 + 버튼)
- ✅ 스티커 개별 업로드 (각 팩의 "스티커 추가" 버튼)
- ✅ 스티커 개별 삭제 (각 스티커의 X 버튼)
- ✅ 스티커팩 전체 삭제 (휴지통 아이콘)
- ✅ 실시간 Firestore 동기화
- ✅ 업로드 진행률 표시

## 📲 설치 방법

### 방법 1: 기존 앱 삭제 후 새로 설치 (권장)
1. **기존 QRChat 앱 완전 삭제**
   - 설정 → 앱 → QRChat → 삭제
   - "앱 데이터도 삭제" 체크 ✅

2. **새 APK 설치**
   - `qrchat_v1.0.16.apk` 다운로드
   - APK 파일 클릭하여 설치
   - "알 수 없는 출처" 허용 필요

3. **로그인**
   - 앱 실행
   - 계정 로그인
   - QKEY 잔액 확인: **60 QKEY** ✅

### 방법 2: 덮어쓰기 설치
1. APK 파일 다운로드
2. 기존 앱 위에 설치
3. 로그아웃 후 다시 로그인 (잔액 갱신 위해)

## ✅ 설치 후 확인 사항

### 1. 버전 확인
- 로그인 화면 하단: **Version 1.0.16+116**
- 프로필 → 설정 → About: **Version 1.0.16+116**

### 2. QKEY 잔액 확인
- 프로필 탭 → QKEY 포인트 카드
- 표시 잔액: **60 QKEY** (바보바보님 기준)
- 하단 텍스트: "채팅 5분마다 **2 QKEY** 적립" ✅

### 3. 거래 내역 확인
- QKEY 포인트 카드 → "거래 내역" 버튼
- 모든 항목: **+2 QKEY** 표시
- 총 30건 × 2 QKEY = 60 QKEY 확인

### 4. 스티커팩 관리 확인
- 프로필 → ⚙️ 설정
- 스크롤 다운
- "About" 아래 "스티커팩 관리" 메뉴 확인
- 클릭하여 관리 화면 진입
- 스티커 추가/삭제 기능 테스트

## 🎯 주요 기능 위치

### QKEY 관련
- **잔액 확인**: 프로필 탭
- **거래 내역**: 프로필 → QKEY 카드 → "거래 내역"
- **출금 신청**: 프로필 → QKEY 카드 → "출금 신청"

### 스티커팩 관리
- **관리 화면**: 프로필 → ⚙️ → "스티커팩 관리"
- **새 팩 생성**: 관리 화면 → 우측 상단 + 버튼
- **스티커 추가**: 각 팩 확장 → "스티커 추가" 버튼
- **스티커 삭제**: 각 스티커의 X 버튼 클릭
- **팩 삭제**: 각 팩의 휴지통 아이콘 클릭

## ⚠️ 문제 해결

### Q: QKEY 잔액이 여전히 300으로 표시됩니다
A: **로그아웃 후 다시 로그인**하세요. 캐시된 데이터가 사용되고 있습니다.

### Q: 스티커팩 관리 메뉴가 보이지 않습니다
A: 
1. 프로필 → ⚙️ 설정 클릭
2. 설정 화면에서 **아래로 스크롤**
3. "About" 메뉴 바로 아래에 있습니다
4. 보라색 아이콘(📚)과 함께 "스티커팩 관리" 표시

### Q: 앱 버전이 1.0.16으로 표시되지 않습니다
A: 
1. 기존 앱 완전 삭제 (앱 데이터 포함)
2. 새 APK 다운로드 (캐시 삭제 또는 시크릿 모드)
3. 재설치
