# QR Chat Application

## Description
QR Chat is an Android application for chat communication using QR codes.

## Files
- `app-release.apk` - Android application package (70MB)

## Installation
1. Download the APK file
2. Enable installation from unknown sources on your Android device
3. Install the APK
4. Launch the QR Chat application
