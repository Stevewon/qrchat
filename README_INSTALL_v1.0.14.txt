   - ListTile 형태로 더 눈에 잘 보이게 개선
   - 보라색 스티커 아이콘으로 쉽게 인식

📊 QKEY 수정 내역
---------------
### 수정 전
- 10 QKEY 거래: 7개
- 2 QKEY 거래: 58개
- **문제**: 기존 거래 내역이 10으로 남아있음

### 수정 후 (v1.0.14)
- **2 QKEY 거래: 65개 (100%)** ✅
- 10 QKEY 거래: 0개
- 영향받은 사용자: 4명
  - 1770301221720: 60 QKEY
  - 1770363136308: 14 QKEY
  - 1770434260975: 10 QKEY
  - 1770305983347: 4 QKEY

### 수정 작업
1. 모든 10 QKEY 거래를 2 QKEY로 변경
2. balanceAfter 값 재계산 (기존 잔액 - 8)
3. 각 사용자의 현재 잔액 재계산 및 업데이트
4. 거래 내역 일관성 확인

🎨 스티커팩 관리 기능
------------------
### 접근 방법
1. 앱 실행 → 로그인
2. 하단 네비게이션 → **프로필** 탭
3. 우측 상단 **설정** 아이콘 (⚙️)
4. 조금 스크롤 → **"스티커팩 관리"** 메뉴
   - 🟣 보라색 스티커 아이콘
   - "About" 바로 아래 위치
   - 서브타이틀: "스티커 추가, 삭제 및 팩 관리"

### 기능 목록
✅ **새 스티커팩 생성**
   - 우측 상단 [+] 버튼
   - 팩 이름 입력

✅ **스티커 추가**
   - 팩 펼치기 → "스티커 추가" 버튼
   - 갤러리에서 이미지 선택
   - 업로드 진행률 표시

✅ **스티커 삭제**
   - 스티커 우측 상단 [X] 버튼
   - 확인 후 Storage + Firestore 삭제

✅ **스티커팩 삭제**
   - 팩 제목 우측 휴지통 버튼
   - 모든 스티커 및 데이터 삭제

✅ **실시간 동기화**
   - Firestore Stream 사용
   - 변경사항 즉시 반영

📱 설치 방법
----------
1. **기존 QRChat 앱 삭제 (필수)**
   - v1.0.12 또는 v1.0.13을 사용 중이라면 삭제
   
2. **v1.0.14 APK 설치**
   - qrchat_v1.0.14.apk 다운로드
   - "알 수 없는 출처" 권한 허용
   - 설치 진행

3. **앱 실행 및 확인**
   - 로그인
   - 프로필 → QKEY 거래 내역 확인
   - 설정 → 스티커팩 관리 메뉴 확인

🔍 변경사항 확인 방법
------------------
### QKEY 확인
1. 프로필 탭 → "거래 내역" 버튼
2. 모든 적립 거래가 **2 QKEY**로 표시되는지 확인
3. 잔액이 정확한지 확인

### 스티커팩 관리 확인
1. 프로필 탭 → 설정 아이콘 (⚙️)
2. "About" 메뉴 아래에 "스티커팩 관리" 메뉴 확인
3. 보라색 아이콘과 설명 확인
4. 탭하여 스티커팩 관리 화면 진입
