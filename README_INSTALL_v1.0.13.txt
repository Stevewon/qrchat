QRChat v1.0.13 - 스티커팩 관리 UI 개선
========================================

빌드 날짜: 2026-02-14
빌드 버전: v1.0.13 (Build 113)

🎨 주요 개선사항
--------------
✅ 스티커팩 관리 메뉴를 더 눈에 잘 띄는 위치로 이동
   - 프로필 → 설정 → "스티커팩 관리" 메뉴 (About 아래)
   - ListTile 형태로 변경하여 다른 설정 메뉴와 일관성 유지
   - 서브타이틀 추가: "스티커 추가, 삭제 및 팩 관리"

🎯 스티커팩 관리 기능 (v1.0.11부터 포함)
-----------------------------------
✅ 스티커팩 생성
   - 새로운 스티커팩 추가
   - 팩 이름 지정

✅ 스티커 개별 업로드
   - 갤러리에서 이미지 선택
   - Firebase Storage 자동 업로드
   - 업로드 진행률 표시

✅ 스티커 개별 삭제
   - 각 스티커의 X 버튼으로 삭제
   - 확인 다이얼로그
   - Storage + Firestore 동시 삭제

✅ 스티커팩 삭제
   - 전체 팩 삭제
   - 모든 스티커 및 데이터 삭제

✅ 실시간 동기화
   - Firestore Stream 사용
   - 변경사항 즉시 반영

📱 사용 방법
----------
1. 앱 실행 후 로그인
2. 하단 네비게이션 바에서 "프로필" 탭 선택
3. "설정" 아이콘 클릭 (톱니바퀴 모양)
4. 스크롤하여 "스티커팩 관리" 메뉴 찾기
   → 🟣 보라색 스티커 아이콘으로 표시됨
5. 탭하여 스티커팩 관리 화면으로 이동

🎨 스티커팩 관리 화면 사용법
-------------------------
**새 팩 생성**
- 우측 상단 [+] 버튼
- 팩 이름 입력 후 생성

**스티커 추가**
1. 팩 이름 탭하여 펼치기
2. "스티커 추가" 버튼 클릭
3. 갤러리에서 이미지 선택
4. 업로드 완료 대기

**스티커 삭제**
- 스티커 우측 상단의 [X] 버튼 클릭
- 확인 다이얼로그에서 "삭제" 선택

**팩 삭제**
- 팩 제목 우측의 휴지통 버튼 클릭
- 확인 다이얼로그에서 "삭제" 선택
