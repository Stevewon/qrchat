QRChat v1.0.11 Android APK 설치 가이드
=========================================

파일 정보:
- 파일명: qrchat_v1.0.11.apk
- 크기: 27.9 MB
- 최소 Android 버전: 5.0 (API 21)

주요 변경사항 (v1.0.11):
✅ 스티커팩 관리 기능 추가 (관리자용)
   - 스티커팩 목록 조회
   - 스티커 개별 업로드 (갤러리에서 선택)
   - 스티커 개별 삭제
   - 새 스티커팩 생성
   - 스티커팩 삭제
   - 실시간 업로드 진행률 표시

🎨 스티커팩 관리 화면 접근:
   - 프로필 화면 > "스티커팩 관리" 버튼 클릭
   - 보라색 버튼으로 표시됨

📌 스티커팩 관리 기능:
   1. 스티커팩 생성: 우상단 "+" 버튼
   2. 스티커 추가: 스티커팩 펼친 후 "스티커 추가" 버튼
   3. 스티커 삭제: 각 스티커 우상단 X 버튼
   4. 스티커팩 삭제: 스티커팩 우측 휴지통 아이콘

기존 기능 유지 (v1.0.10):
✓ QKEY 적립량 2개 (5분당)
✓ 스티커팩 탭 구분 (카카오톡 스타일)
✓ 그룹 채팅 이미지/스티커 크기 일관성
✓ 동영상 썸네일 생성
✓ 친구 목록 가나다 정렬
✓ QR 주소로 닉네임/비밀번호 찾기
✓ 회원가입 닉네임 중복 체크
✓ 차단 사용자 로그인 방지

설치 방법:
1. 휴대폰에서 이 ZIP 파일을 다운로드하여 압축 해제
2. qrchat_v1.0.11.apk 파일 실행
3. "출처를 알 수 없는 앱 설치" 권한 허용 (필요 시)
4. 설치 진행

⚠️ 중요 사항:
- Firebase Storage 규칙이 read/write 허용으로 설정되어 있어야 합니다
- 스티커 업로드를 위해 Storage 쓰기 권한 필수
- 규칙 설정 위치: https://console.firebase.google.com/project/qrchat-b7a67/storage/rules
- 규칙 예시:
  ```
  rules_version = '2';
  service firebase.storage {
    match /b/{bucket}/o {
      match /{allPaths=**} {
        allow read, write: if true;
      }
    }
  }
  ```

🎨 스티커팩 관리 가이드:

1️⃣ 새 스티커팩 만들기:
   - "스티커팩 관리" 화면 우상단 "+" 클릭
   - 스티커팩 이름 입력 (예: "동물 친구들")
   - "만들기" 클릭

2️⃣ 스티커 추가하기:
   - 스티커팩 클릭하여 펼치기
   - "스티커 추가" 버튼 클릭
   - 갤러리에서 이미지 선택
   - 스티커 이름 입력 (예: "웃는 고양이")
   - 자동 업로드 (진행률 표시)

3️⃣ 스티커 삭제하기:
   - 삭제할 스티커 우상단 X 버튼 클릭
   - 확인 다이얼로그에서 "삭제" 클릭

4️⃣ 스티커팩 삭제하기:
   - 스티커팩 우측 휴지통 아이콘 클릭
   - 확인 다이얼로그에서 "삭제" 클릭
   - ⚠️ 스티커팩 내 모든 스티커가 함께 삭제됨

💡 스티커 업로드 팁:
   - 권장 이미지 형식: PNG (투명 배경 지원)
   - 권장 이미지 크기: 512x512px ~ 1024x1024px
   - 파일 크기: 1MB 이하 권장
   - 애니메이션: GIF, WebP 지원

📊 스티커팩 자동 반영:
   - 스티커 추가/삭제 시 즉시 앱에 반영
   - 앱 재시작 불필요
   - 실시간 Firestore 동기화
